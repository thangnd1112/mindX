Cho đoạn code sau, sau đó biểu diễn các cách để truy cập được các thuộc tính bên trong object đó

                let person = {
                    name: 'Bob',
                    occupation: 'web developer',
                    hobbies: 'painting',
                };

Cách 1 - Truy cập bằng dấu chấm .

Cách 2 - Truy cập bằng dấu ngoặc vuông

Cách 3 - Sử dụng Object destructuring

Result

Cho object obj, hãy tạo một array chứa key của tất cả các property của obj và đều viết hoa

                let obj = {
                    b: 2,
                    a: 1,
                    c: 3,
                };

Ý tưởng

  • Dùng method Object.keys để lấy các key của Object, sau đó lưu vào 1 biến, biến này sẽ lưu trữ 1 mảng là kết quả trả về từ method trên.
  • Dùng vòng lặp for hoặc method forEach để duyệt qua từng phần tử của mảng, với mỗi lần lặp thì dùng method toUpperCase để in hoa.

Result

Tạo một object mới có tên **family** và sử dụng Object **myFamily** như là prototype của object đó

Khởi tạo hàm tạo Family

            function Family(quantity,member,location) {
                this.quantity = quantity,
                this.member = member,
                this.location = location
            }

Tạo object mới từ hàm tạo Family

            let newFamily = new Family(myFamily.quantity, myFamily.member, myFamily.location);
